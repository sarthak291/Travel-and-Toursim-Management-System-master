# Travel and Tourism Management System

A **desktop-based Travel and Tourism Management System** built in **Java** to automate and manage core travel agency operations such as **user registration, package browsing, hotel booking, package booking, and personal detail management**. This system provides a simple GUI for users and helps agencies manage travel related tasks efficiently.
